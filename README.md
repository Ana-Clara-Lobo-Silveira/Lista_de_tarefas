# Lista de tarefas

Um projeto criado com fins educativos e para treino de linguagem. Consiste em uma lista de tarefas, sujeita a alterações, executada no próprio terminal.

## Objetivos
* Prática da estrutura e funcionamento do "While".
* Estudo e treino referente à lógica de programação.

## Tecnologias
* Linguagem: Python

## Instalacao e Uso
1. Clone o repositório ou baixe os arquivos.
2. O projeto não exige instalações externas, basta apenas executar o arquivo "main.py".
